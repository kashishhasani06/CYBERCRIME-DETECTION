 AI-Based Cybercrime Detection System

 Project Overview

This project is a simple and interactive cybercrime detection system developed using Python. It is designed to help users identify potential online threats such as phishing messages, malicious links, and suspicious phone numbers. The system combines basic artificial intelligence concepts with rule-based analysis to provide quick and understandable results.



 Features

*AI Message Scam Detector**
  Detects phishing messages using a word-based scoring model.

*Malicious Link Detector**
  Analyzes URLs and identifies suspicious patterns like unsafe protocols and keywords.

*Phone Number Analyzer**
  Checks number validity, estimates operator and region, and detects fraud patterns.

*Menu-Driven Interface**
  Allows users to easily select and use different tools.



 How It Works

* The message detector uses a simple AI model that assigns scores to words based on training data.
* The URL detector uses rule-based logic to identify risky patterns.
* The phone analyzer evaluates number structure and detects suspicious behavior while estimating operator details.



 Technologies Used

* Python
* Basic Machine Learning (word scoring)
* String Processing
* Conditional Logic



 How to Run

1. Open the Python file in any editor (VS Code recommended)
2. Run the program
3. Select a tool from the menu:

   * 1 → Message Detector
   * 2 → Link Detector
   * 3 → Phone Analyzer
   * 4 → Exit
4. Enter input and view results



 Limitations

* Uses a small dataset for message detection
* Operator detection is approximate due to mobile number portability
* URL analysis is rule-based (no real-time database)
